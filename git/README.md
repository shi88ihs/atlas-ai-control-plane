# Atlas Control Plane Git Helper
 
 This module provides Git awareness for the Atlas Control Plane. Future commands will be integrated into the `control-plane` CLI wrapper to manage version control operations.

## Future Commands

- `control-plane git status`: View repository, branch, last commit, remote, and working tree status.
- `control-plane git commit`: Stage all safe changes and create a commit.
- `control-plane git push`: Push local commits to the remote repository.
- `control-plane git pull`: Pull latest changes from the remote repository.
- `control-plane git log`: View recent commit history.
